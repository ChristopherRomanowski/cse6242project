DESCRIPTION:
This package represents the experiments and the final code to conduct our model for predicting the outcome and select features about the results of future game (final score, total yards by each team)

INSTALLATION: - How to install and setup your code
Using python 3.6+
Must install requirements.txt
"pip install -r requirements.txt"

EXECUTION:
The code to train and produce the final game predictions is given by
"python build_final_results.py"
